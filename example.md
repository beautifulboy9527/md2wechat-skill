# 欢迎使用 md2wechat

这是一篇示例文章，帮助你快速了解 md2wechat 的排版效果。

## ✨ 功能亮点

**md2wechat** 可以帮你：

- 用 Markdown 写公众号文章
- 自动生成精美排版
- AI 生成封面图
- 一键上传到草稿箱

## 📝 Markdown 语法支持

### 代码块
```python
print("Hello, md2wechat!")
```

### 表格
| 功能 | 状态 |
|------|------|
| 排版 | ✅ |
| 生图 | ✅ |
| 上传 | ✅ |
